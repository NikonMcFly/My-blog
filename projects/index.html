Go-Weather is Forecast.io API that gives the user a heads up on what the weather may look like before they decide to travel. It is built with React/Redux on the front-end and Golang on the back-end.

Bitcoin-Ticker is a web app powered by Bitcoin Average and Local Bitcoins to show where you can buy and sell bitcoins around the United States at an given price. This project will be built with React Redux framework on the front-end with Webpack as the build system for integrating ES6 and Golang on the back-end.
